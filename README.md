# 燃灯寺公墓静态站点

这是一个可直接部署的纯静态网站版本，当前仅使用燃灯寺相关的实景素材整理完成，适合：

- 推送到 GitHub 做版本管理
- 接入 Cloudflare Pages 做自动发布
- 后续继续补充墓型、价格、地图、企业介绍、表单等内容

## 本地文件

- `index.html`：站点首页
- `404.html`：静态 404 回退页
- `assets/css/styles.css`：页面样式
- `assets/js/main.js`：tab 切换和图片预览交互
- `sucai/`：新增实景墓碑图片、墓区环境图片和航拍视频
- `_headers`：Cloudflare Pages 缓存与安全响应头
- `.nojekyll`：避免静态托管时对特殊文件名产生干扰

## 当前整合内容

- 实景墓碑展示
- 墓区环境照片展示
- 园区航拍视频播放
- 一键拨号与地址信息

## Cloudflare Pages 建议

这是零依赖静态站点，不需要构建命令。后续连接 GitHub 后，优先按“静态站点 / 无框架”的方式接入即可。

可按下面思路操作：

1. 把当前目录推到 GitHub 仓库。
2. 在 Cloudflare Pages 中连接这个 GitHub 仓库。
3. 选择静态站点方案，不安装依赖。
4. 构建命令留空，输出目录使用根目录 `/`。
5. 让站点根目录直接作为发布内容。
